TravelAgency Customers API

Simple API for maintaining customers of Travel Agency company

Access

To access users data, an API key is required. Please, contact our system administrator.

Usage

API provides information about users and their trips, data are presented in JSON format. Accessible endpoints are:

HTTP Method Endpoint Description
GET / This homepage
GET /customers Get all customers
GET /customer/<string:username>Get specific customer
POST /customer/<string:username>Create new customer
PUT /customer/<string:username>Update customer
DELETE /customer/<string:username>Delete customer
GET /trips Get all trips
GET /trip/<int:trip_id> Get specific trip
POST /trip/<int:trip_id> Create new trip
PUT /trip/<int:trip_id> Update existing trip
DELETE /trip/<int:trip_id> Delete trip
